Guide rapide (dépannage)

  1. Assurez-vous d’avoir la dernière version (App Store / TestFlight).
  2. Vérifiez l’autorisation de Localisation : iOS → Réglages → Confidentialité → Localisation → RunPacer → « Pendant l’utilisation ».
  3. Activez le GPS/Position et testez à l’extérieur (meilleure réception).
  4. Si Strava est connecté, essayez de déconnecter/reconnecter.
  5. Redémarrez l’app (et si besoin, l’iPhone).

FAQ

Le suivi GPS ne démarre pas / reste à 0 km

Vérifiez l’autorisation Localisation (voir Guide rapide). Lancez la course à l’extérieur et attendez l’accroche du signal. Évitez le mode Économie d’énergie qui peut limiter la précision.

Connexion Strava

La connexion est optionnelle. Si la synchro échoue, déconnectez puis reconnectez votre compte depuis l’app. Assurez-vous d’autoriser RunPacer dans Strava.

Instructions vocales

Activez l’option dans Profil → « Instructions vocales ». Montez le volume et vérifiez que le mode silencieux n’est pas activé.

Suppression de mes données

Les données d’entraînement sont stockées localement sur votre appareil. Pour les supprimer, désinstallez l’app. Si vous avez synchronisé avec Strava, gérez la suppression directement depuis votre compte Strava.
